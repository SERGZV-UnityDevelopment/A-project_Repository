# A_Project
Это мой первый проект работу над которым я вёл с 2013 по 2015 год.
В нём я хотел сделать что - то на подобе игры готики. Но понял что один человек такую игру будет делать десятки лет. Успел сделать человека за которым следует камера. Камеру можно приблизить и отдалить. Камера сама приближается к персонажу если врезается в стену. Герой может подбирать предметы которые автоматичесски сортируются в инвентаре по категориям. Если переключится на категорию оружие то инвентарь будет показывать толко оружие. Так же игрок может выкидывать предметы для которых создана модель.
Имеется переключение на ходьбу и бег.
