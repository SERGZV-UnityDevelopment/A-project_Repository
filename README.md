# A_Project

Это мой первый проект над которым я работал где то в 2014 году.
В нём я хотел сделать что - то на подобе игры Gothic. Но понял что один человек такую игру будет делать десятки лет.
Большая часть времени ушла на создание карты и моделирования объектов для неё. БольшУю часть кода пришлось переписать чтобы сборка работала на версии Unity 2019.3 с интерфейсом UI вместо старого GUI.
Успел сделать человека за которым следует камера. Камеру можно приблизить и отдалить. Камера сама приближается к персонажу если уперается в стену. Герой может подбирать предметы которые автоматичесски сортируются в инвентаре по категориям. Если переключится на категорию оружие то инвентарь будет показывать толко оружие. Так же игрок может выкидывать предметы для которых создана модель.
Имеется переключение на ходьбу и бег.
